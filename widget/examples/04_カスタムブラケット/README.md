# 03. 機械部品 — オーダーメイドブラケット / Custom Sheet Metal Bracket

## 概要
**パラメータ例:** 取付穴ピッチ・プレート厚・折り曲げ角度
設備改造時に既製品が合わない場面が多い。シンプルな折り曲げブラケットをパラメータだけで注文できるようにすれば受注単価を下げつつ量をこなせる。
