# 04. 電子機器筐体 — カスタムエンクロージャ / Custom Electronics Enclosure

## 概要
**パラメータ例:** PCB外形サイズ・コネクタ穴種類・通気孔パターン・塗色
小ロット電子機器の筐体は毎回設計が必要。基板寸法を入力するだけで穴位置が自動配置されるコンフィギュレータは開発工数を大幅削減する。
