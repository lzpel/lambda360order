# 11. 化学・石油プラント — 配管サポートクランプ / Insulated Pipe Support Clamp

## 概要
**パラメータ例:** 配管外径・断熱材厚・クランプ幅・架台ボルトピッチ
プラント配管は口径と断熱仕様の組み合わせが無数にある。断熱材厚を入力するだけでクランプ内径が自動計算される構成は調達担当者の負担を下げる。
