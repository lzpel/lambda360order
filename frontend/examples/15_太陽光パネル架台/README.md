# 14. 再生可能エネルギー — 太陽光パネル架台 / Solar Panel Mounting Structure

## 概要
**パラメータ例:** パネル枚数・傾斜角・積雪荷重クラス・アンカー基礎ピッチ
屋根形状と設置条件によって架台形状が変わる。パネル枚数を入力すると全体フレームが StretchNode で自動伸長し、傾斜角が RotateNode に反映される。
