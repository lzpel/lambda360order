# 07. 農業機械 — アタッチメントブラケット / Tractor 3-Point Hitch Attachment Bracket

## 概要
**パラメータ例:** トラクター機種（3点リンク規格）・アーム長・取付角度
トラクターへの農具取り付けブラケットは機種依存が強い。機種を enum で選ぶだけで穴ピッチと強度計算が変わる構成にできる。
