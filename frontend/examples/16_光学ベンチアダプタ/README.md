# 15. 実験・研究機器 — 光学ベンチアダプタ / Optical Bench Post & Adapter

## 概要
**パラメータ例:** ベースレール規格（M6/M4）・取付高さ・ポスト径・素材
光学実験は試行ごとに光路が変わり、専用アダプタが必要になる。研究者自身がブラウザから発注できるフローはプロトタイピング速度を上げる。
