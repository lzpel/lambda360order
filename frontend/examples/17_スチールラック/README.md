# 16. 家具・什器 — スチールラック / Steel Storage Shelving Rack

## 概要
**パラメータ例:** 段数・棚板幅・棚板奥行き・耐荷重クラス・塗色
倉庫や工場内の収納棚は設置スペースと収納物に合わせた寸法が必要。段数と幅を入力すると柱高と棚板位置が自動計算され、3D で確認してから発注できる。
