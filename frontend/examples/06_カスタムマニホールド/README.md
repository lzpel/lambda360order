# 05. 配管・流体機器 — カスタムマニホールド / Custom Hydraulic / Pneumatic Manifold

## 概要
**パラメータ例:** ポート数・ポート間ピッチ・継手規格（Rc/NPT）・材質
油圧・空圧システムの配管集約部品。ポート数に応じて本体長を StretchNode で伸ばし、SubtractNode で穴を追加する構成が直感的にマッピングできる。
