# 12. 船舶・海洋 — デッキフィッティング / Marine Deck Fitting

## 概要
**パラメータ例:** 甲板厚・ボルト配置・素材（SUS316/アルミ）・表面処理
船体ごとに異なる甲板厚と取り付け位置に合わせたフィッティング部品。錆耐性と強度を材質 enum で選べる購買体験が B2B 発注に適合する。
