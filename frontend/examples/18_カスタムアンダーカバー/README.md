# 17. 自動車アフターマーケット — カスタムアンダーカバー / Custom Underbody Skid Plate Cover

## 概要
**パラメータ例:** 車種（enum）・エンジン開口形状・固定クリップ位置・素材（スチール/アルミ）
車種ごとに形状が固定されているが素材・開口・クリップ位置だけ変えたい需要がある。車種 enum で基本形状を選び、開口をパラメータで微調整するセミカスタム構成。
